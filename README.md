meteor-html5shivcdn
===================

CDN edition of html5shiv / html5shim packaged for Meteor.

-   Homepage of html5shiv: [html5shiv][] \<<https://code.google.com/p/html5shiv//>\>

HowTo:

> Just meteor add mirageglobe:html5shiv

  [html5shiv]: https://code.google.com/p/html5shiv/
